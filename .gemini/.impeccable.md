## Design Context

### Users
Jogadores casuais buscando uma partida rápida de Jogo da Velha, seja localmente (PVP) ou contra a CPU (PVE).

### Brand Personality
Suave, Moderno, Agradável.
- **3-word personality**: Sophisticated, Playful, Clean.

### Aesthetic Direction
Design limpo com toques de sofisticação (Glassmorphism). Uso de desfoque de fundo e bordas sutis para dar profundidade sem poluir visualmente.

### Design Principles
1. **Coesão Cromática**: Usar estritamente a paleta base (#43115B, #dcbf3f, #72cff9).
2. **Hierarquia Clara**: Botões de ação com contraste e estados de hover bem definidos.
3. **Responsividade Fluida**: Adaptação perfeita de smartphones a desktops usando `clamp()` e `container queries`.
4. **Micro-interações**: Transições suaves (como o `fadeIn` do menu) entre os estados do jogo.

### Tech Stack Constraints
- React (TypeScript)
- CSS Modules
- Vanilla CSS para animações e efeitos
